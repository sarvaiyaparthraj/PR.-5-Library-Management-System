# 📚 Library Management System (C++ OOP Project)

A console-based **Library Management System** developed using **C++ and Object-Oriented Programming (OOP)** concepts.  
This project allows users to manage different library items such as:

- 📖 Books
- 🎬 DVDs
- 📰 Magazines

The system uses advanced OOP concepts like:
- Inheritance
- Polymorphism
- Abstract Classes
- Virtual Functions
- Dynamic Memory Allocation
- Exception Handling

# 📌 Project Overview

The Library Management System provides a menu-driven interface where users can:

✅ Check Out Items

This project is designed to simulate a real-world library system while demonstrating strong OOP principles.
